錢芝萍 學員申請書

"人到中年，燃起學習熱情！"

WeHelp的導師們大家好，我是錢芝萍，畢業於元智大學資訊傳播系，現年 40 歲、有兩個小孩，目前在自家音響工程公司擔任業務經理的工作。由於大學所學主要偏重在設計與行銷領域，在這20年的職涯裡，畢業後的前10年，我在外面公司擔任過平面 設計師、行銷企劃專員等職務；後10年開始，在爸爸的公司協助內外勤業務，工作內容從工地場勘、AutoCAD製圖到合約議價等，內容可以說包山包海。

申請動機

2017年開始接觸西洋哲學，這為我開啟了一扇窗，這才發現學習是一件非常快樂的事情，以前排斥的數學、科學、邏輯等內容都讓人充滿了樂趣。

自此我如上癮般愛上了閱讀與學習，從去年開始接觸一些應用數學、科學的書籍，到了今年看到了一本書《思考的演算》，我覺得整個人燃起了程式魂，裡面提到X光 程式的運作原理、濾波器的操作邏輯，都深深吸引了我想更了解許多事物背後的程式邏輯。

我也開始反思工作的意義，畢竟工作占了快1/3的寶貴人生，我是不是能做一份更有意義、結合自己的興趣、又能常常學習新事物的工作呢？ 我得到的答案是，我想轉職成為 程式設計師。

曾經做過哪些軟體工程技術相關的學習？若有作品請分享給我們。

大學資訊傳播系的時期，學過 HTML、Flash、Dreamweaver，算是跟網站設計相關的軟體。

曾經在2017年育嬰假期間，完成 Coursera 三門 Python 課程，之後中斷到今年2024年1月，我下定決心為轉職做準備，在 Udemy 自學 Python、Web development的課程、在 youtube 自學彭彭課程，並且在 kaggle 學習資料分析與清理。

因為想增加個人專案經驗，我在今年7月參與中央研究院舉辦的第一屆 TBIA 生物開放資料數據松競賽，很榮幸以鳥類新聞媒體偏好分析得到最佳人氣獎，報名的當下不太熟悉 爬蟲與資料清理等技術，在一個月內靠著彭彭的爬蟲教學、圖書館借來的書籍，最終使用Python進行爬蟲與資料整理，使用Tableau作圖表視覺化完成專案。

連結我的作品

如果參與這個訓練，會怎麼安排學習時間？

參與這個訓練前，我會先從自家公司離職，全心投入於 WeHelp Bootcamp 的訓練，學習規劃將如下：

  1. 自我準備：目前有參與 Udemy 線上程式共讀會，除了跟著社群學員一起複習 Python 的基本操作，也會在開訓前複習完線上彭彭與 Udemy 購買的課程。
  2. 時間規劃：由於全職投入訓練，因此期許自己在每周一收到任務後，盡量在周間就把任務完成，剩下的時間可以作一些任務延伸的個人練習，或找相關書籍加強實作， 也能規劃轉職面試用的作品。
  3. 社群交流：我將參加實體聚會，並且積極跟大家交流心得與困難，與第六屆的大家一起順利畢業與轉職，未來各自在業界發展也能持續這份友誼。
  4. 作品累積：預計完善自己的 github，作為未來求職的敲門磚，如果有數據分析，或是程式相關的比賽，也會視 Bootcamp 學習狀況，在行有餘力時報名挑戰自己。

軟體技術日新月異，如何確定選擇投入的領域是正確有回報的？

老實說，我無法知道這個投入是否能有回報。

我只能知道，當一個人對某項事物有熱情、有興趣，不管這個領域未來發展好不好，他都能應用所學，不管是找相關的工作，或是自己有能力開發新網站與APP。

請描述一件產生明顯負面情緒的經歷，如何處理該情緒？

我從今年2月開始加入一個書籍共讀會，選讀非常廣泛 10月是《約翰彌爾自傳》、9月是《博爾赫斯全集》、8月《窮查理的普通常識》，每月社群裡有一位擔 任主持人、分享一本書，我在7月和10月皆自願擔任主持人。

每次線上會議開始前，其實會有點負面情緒，混和著緊張與不安，自問為何我要自找麻煩，當個與會者不是輕鬆多了？別人會不會覺得我主持得很糟糕？

通常我都會要自己冷靜下來，理性上來說，因為我做的這件事我不夠熟悉，所以會有點焦慮，既然我有認真準備了，就好好享受這個主持的過程。另外，大家來參加這個 活動是因為對書感興趣，而不是來評判我，這樣想過後心態上就會平穩多了。

關於這份申請網頁，分享一個開發時的技術心得。

看完彭彭老師的 Front End 網頁前端工程教學後，會特別注意網頁必須在不同平台，都能夠清楚顯示這件事，因此在開發的過程中，會一直測試網頁是否因為 網頁大小、加入 Javascript 有任何版面不和諧的問題。

當我使用 innerHTML 來切換不同頁面時，首頁與其他分頁會有網頁偏移的問題，我發現可能是因為垂直捲軸造成的問題，後來在 CSS 中加入 overflow-y: overlay， 讓捲軸疊加在內容上面，而不佔用寬度，避免版面產生位移。

如何看待自身工作和整個社會群體的連結關係？

以哲學家的角度來看這個問題，他們會認為自身的工作，要能對人類世界的進步帶來助益。牛頓說：「如果我能看得更遠， 那是因為站在巨人的肩膀上。」我認為以程式設計師來說，從 github 的社群能看得出來，個人與社群的連結性，我們每個人都能對社群提供微小的貢獻。

其他想要對我們說的事情？

當您在看這份申請書時，可能會疑惑我是否有｢中年轉職｣的焦慮與迷惘？我認為中年轉職不困難，困難的是一個人是否勇於面對既有價值觀，試著突破既有年齡與非本科的框架。當我們認同漫漫人生旅途中，一切皆學無止盡，年紀不會是學習或是轉職的困難時，困難將會成為一種前進的動力。
